Computer Organization (Embedded Systems) | Magic Mirror
For our final project, we would like to create a smart mirror using both a raspberry pi and the FPGA Board. The Mirror will feature visual data from a camera as the input, and will use that visual data to send through the network to an application on the web. The mirror will also pull and display data such as weather, stocks, calendar, etc. overlayed on top of the "reflection" giving the appearance of a single image with useful widgets seamlessly integrated with the reflection. To do this, we will use a two-way mirror and a monitor(with webcam) connected to the raspberry pi, which will send visual data to the FPGA for network and application processing.
